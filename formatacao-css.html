Início Estilos em Cascata Formatação CSS Box Model

O que é css ?

CSS é uma linguagem baseada em regras. — Você define regras especificando grupos de estilo que devem ser aplicados para elementos particulares ou grupos de elementos na sua página web.

Exemplos de Formatação

Este texto está formatado com uma cor azul e tamanho de fonte de 18px.

Este texto está centralizado e com espaçamento entre linhas ajustado.

O estilo de Fonte em Vermelho Fica asssim

Estilo de fonte em vermelho

O codigo usado para isso foi:

''color: red;''

O estilo de Fonte larga fica assim

Texto largo

O codigo usado para isso foi:

''font-size: largue;

O estilo de Fonte para a Esquerda fica assim

Texto alinhado para a esquerda.

O codigo usado para em CSS isso foi:

.esquerda { text-align: left; color: green; }

O estilo de Fonte para a Direita fica assim

Texto alinhado para a direita.

O codigo usado para em CSS isso foi:

text-align: right; color: blue; }

O estilo de Fonte para o Centro fica assim

Texto alinhado no centro.

O codigo usado para em CSS isso foi:

text-align: center; color: purple; }
